Le projet de système de régulation de température à l'aide d'une API vise à développer une solution intelligente permettant de contrôler et de maintenir la température optimale dans divers environnements, tels que les maisons, les bureaux ou les installations industrielles. En utilisant une sonde pour mesurer la température ambiante en temps réel, le système envoie les données à une API centralisée. L'API analyse ces informations et ajuste automatiquement les dispositifs de chauffage, le système sera relié à un thermoplongeur.
